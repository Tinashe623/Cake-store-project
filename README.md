## 📖 About

**Tarie Cakes** is a feature-rich bakery storefront web application that allows customers to explore a curated catalog of beautifully designed cakes, place custom orders, and learn about the bakery's story and process. Built with a focus on aesthetics and usability, the site delivers a seamless experience across all devices.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🍰 **Online Cake Browsing** | Explore a rich gallery of cakes organized by category with high-quality images |
| 📝 **Custom Cake Ordering** | Tailor your cake with personalized options — flavor, size, design, and more |
| 📱 **Responsive Design** | Optimized layout for desktop, tablet, and mobile screens |
| 🎨 **Modern UI** | Polished interface with smooth animations and a warm, inviting color palette |
| 🛒 **Shopping Cart** | Add cakes to a cart drawer for a streamlined checkout experience |
| 💬 **Customer Testimonials** | Real reviews from happy customers showcased beautifully |
| ℹ️ **About & How It Works** | Transparent process and bakery story to build customer trust |

## 🚀 Getting Started

### Prerequisites

- **Node.js** >= 18.x
- **npm** or **yarn**

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/cake-store-project.git

# 2. Navigate into the project directory
cd tarie-cakes

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
npm run preview
```
